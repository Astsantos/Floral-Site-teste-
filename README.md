
# Floral — Sistema Web de Gestão Financeira para Floricultura

O **Floral** é uma aplicação web desenvolvida com **HTML, CSS e JavaScript** que simula o gerenciamento financeiro de uma floricultura, permitindo cadastrar projetos, registrar custos, acompanhar pagamentos e visualizar relatórios financeiros de forma prática e intuitiva.
---
## Objetivo do Projeto
Este projeto foi desenvolvido como atividade prática com o objetivo de aplicar conceitos de:
- Estruturação com HTML
- Estilização moderna com CSS
- Manipulação dinâmica com JavaScript
- Persistência de dados com LocalStorage
- Consumo de APIs externas
---
## Funcionalidades:

* Gerenciamento de Projetos
  
- Adicionar novos projetos/obras
- Remover projetos cadastrados

* Controle Financeiro
  
Cadastro de itens de custo contendo:
- Nome do item
- Categoria
- Valor
- Forma de pagamento
- Status do pagamento
- Data
- Observações
## APIs Integradas

* Cotação em tempo real
API utilizada:

AwesomeAPI

Exibe:

- Cotação do dólar
- Cotação do euro

* Busca automática de CEP
